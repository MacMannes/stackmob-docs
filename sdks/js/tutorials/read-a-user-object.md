<h2>Read a User Object</h2>

Let's try to read `User` data for `Bill Watterson`.

```js
var user = new StackMob.User({ username: 'Bill Watterson' });
user.fetch({
	success: function(model) {
	console.debug(model.get('username') + ': ' + model.get('profession'));
	}
});
```
